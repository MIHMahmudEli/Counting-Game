# 🧮 Counting Game: Master the 21

![Version](https://img.shields.io/badge/version-2.0.0-blueviolet)
![License](https://img.shields.io/badge/license-MIT-green)
![Tech](https://img.shields.io/badge/tech-HTML%20%7C%20CSS%20%7C%20JS-orange)

An ultra-modern, high-stakes number game where logic and speed collide. Built with a clean **MVC architecture** and a premium **Glassmorphism UI**, this isn't just a game—it's a challenge against a sassy, unbeatable AI.

---

## 🚀 The Challenge

Count from **1 to 21**. On each turn, you can add **1, 2, or 3** to the total.  
The goal? **Don't be the one to say 21.**

> [!WARNING]
> The computer uses a mathematically perfect strategy. It's smart, it's fast, and it might roast you if you lose.

---

## ✨ Features

- 💎 **Premium UI**: Stunning Glassmorphism design with animated background blobs and high-end typography (`Outfit` & `Inter`).
- 🧠 **Unbeatable AI**: Implements a perfect Nim-sum strategy. It's mathematically optimized to win.
- 💅 **Sassy Personality**: A dynamic "roasting" system that delivers unique, humorous messages upon player defeat.
- 🏗️ **MVC Architecture**: Clean separation of concerns with dedicated Model, View, and Controller layers in pure JavaScript.
- 📱 **Fully Responsive**: Optimized for every screen size, from desktop to mobile, with a strict no-scroll, compact layout.
- ⚡ **Zero Dependencies**: Pure HTML5, CSS3, and Modern JS. No PHP or heavy frameworks required.
