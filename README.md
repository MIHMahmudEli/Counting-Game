# 🧮 Counting Game

A simple and fun **number counting game** built using **HTML, CSS, and JavaScript**.  
Test your luck (and logic) against the computer in a race **not to say 21!**

---

## 🎮 Game Rules

- The game starts by choosing who goes first — you or the computer.  
- Each turn, a player can count **1, 2, or 3** numbers.  
- The numbers continue increasing from **0 → 21**.  
- Whoever says **21 loses** the game.  
- The computer uses a **winning strategy** (multiples of 4 rule) to outsmart you!

---

## ⚙️ Features

- 🧠 Smart computer opponent with perfect strategy.  
- 💡 Dynamic turn indicator.  
- 🔢 Real-time count display with smooth animation.  
- 🚫 Disabled buttons when moves are invalid.  
- 🔄 Restart option after each game.  
- 🎨 Simple, responsive interface using pure HTML, CSS, and JS.
